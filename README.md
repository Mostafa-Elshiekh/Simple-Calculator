# Simple-Calculator
Simple calculator: In short, this project is a calculator that performs basic arithmetic operations (addition, subtraction, multiplication and division) Taking into consideration that it performs these operations on two numbers, each of them has two digits.
"LCD interfacing with Keypad and Microcontroller(AVR-Atmega32)"
